I use AI on some parts of this website. All articles are written by me, and some were proofread using Claude Code, checking things like tone of voice or strength of argument.

Claude is used for creating [artefacts]({{< ref "/artefacts" >}}). I will explicitly list Claude as an author in this case.

Occasionally I'll use Claude Code to modify the theme, though I haven't had much serious luck. It doesn't seem to understand Hugo very well.

I've used Claude to experiment with different font pairings, colour schemes, etc.
